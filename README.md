# Ramos de Lima Advocacia

Site institucional do escritório Ramos de Lima Advocacia.

## Estrutura

```
ramos-de-lima/
├── index.html              # Página principal
├── assets/
│   ├── css/
│   │   └── style.css       # Estilos do site
│   ├── js/
│   │   └── main.js         # Scripts (menu, FAQ, animações)
│   └── images/
│       ├── logo.png        # Logo do escritório
│       ├── dra-layla.jpg   # Foto Dra. Layla
│       └── reginaldo.jpg   # Foto Reginaldo
└── README.md
```

## Como publicar no GitHub Pages

1. Suba todos os arquivos no repositório
2. Vá em Settings → Pages
3. Selecione branch `main` → Save
4. Site disponível em: `https://seu-usuario.github.io/ramos-de-lima`
